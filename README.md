🛒 商场云选合集签到插件说明文档
本插件是专为 Loon 设计的自动化脚本，整合了 猫酷 (Mallcoo) 系统（金桥国际、汇智国际）与 昌宜 (Chamshare) 系统（长泰国际）的自动签到功能。
🌟 主要功能
• 多合一签到：一个任务同时完成三家商场的积分签到。
• 智能名称识别：通知中直接显示商场名称，告别冷冰冰的 ID 数字。
• 精准静默抓包：仅在进入特定页面时触发 Token 更新，不干扰日常使用。
🛠️ 部署指南
1. 插件安装
在 Loon 的 “插件” (Plugins) 页面中，点击右上角 +，选择“从链接安装”，输入您的插件地址：
https://raw.githubusercontent.com/cmkachun/xcuqd/main/xcuqd.plugin
2. 账号抓包（初次使用或 Token 过期时）
插件安装并启用后，请分别执行以下操作以获取登录凭证：
• 金桥国际 / 汇智国际 (猫酷)：
进入小程序，手动点击一次 [签到] 按钮。
• 长泰国际 (昌宜云选)：
只需进入小程序的 [每日签到] 页面即可，无需手动点击签到按钮。


📊 运行报告说明
脚本运行后，会发送一条系统通知，包含以下状态：
• ✅ 签到成功：积分已到账。
• ℹ️ 今日已签：系统检测到今天已经完成过签到，无需重复操作。
• ⚠️ 未获取到 Token：Token 已过期或从未抓包，请参考 [部署指南] 重新抓包。
• ❌ 请求错误/响应异常：通常为网络波动，可尝试手动在“脚本”栏重新运行。
